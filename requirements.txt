# ==============================================================
# requirements.txt  —  WC2026 Match Predictor
# Python 3.13 | MacBook Pro M1 | install inside a venv
#
# Install with:   pip install -r requirements.txt
#
# IMPORTANT: PyTorch is intentionally NOT listed here. It must be
# installed separately so pip resolves the correct Apple-Silicon
# (MPS) build:   pip install torch torchvision torchaudio
# Pinning torch here can pull a wheel without Metal support.
#
# Version style: "~=" is the compatible-release operator.
#   numpy~=2.1.0  means ">=2.1.0, <2.2.0" — patch updates allowed,
#   minor/major locked. This gives reproducibility (a security and
#   science requirement) without freezing you on a broken patch.
# ==============================================================
